# Tricky Towers

This is a simple [tricky towers](http://trickytowers.com/) clone using
[matter.js](http://brm.io/matter-js/). If you do not know the game: It is
basically tetris with physics.

## Controls

-	Left/Right/Up/Down - control falling block
-	P - pause
-	Esc - Restart
